
## JVM (跨语言的平台)

### 特点 

- 一次编译，到处运行
- 自动内存管理
- 自动垃圾回收功能

### 基本概念

JVM是可运行java代码(不仅仅java,Groovy,Scala,JavaScript等)的假想计算机，包括一套字节码指令集，一组寄存器、一个栈、一个垃圾回收、堆和一个存储方法域 。 JVM运行在操作系统上，与硬件没有直接的交互。
![](/img/java/JVM位置.png)

java源文件 通过编译器 可以生成字节码文件(.class文件) 然后通过不同平台的jvm(java虚拟机)中的解释器，编译成电脑能识别的机器码，从而实现一次编译 多次运行

也就是:
- java源文件  -> 编译器  ->字节码文件 
- 字节码文件 -> JVM -> 机器码
